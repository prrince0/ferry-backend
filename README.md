#  Ferry Backend API (Day 1-3)

##  Progress - Day 1

Today I worked on building the core backend setup for the Ferry Management System.

---

##  What I implemented

* Setup Node.js and Express server
* Connected MySQL database
* Configured environment variables using dotenv
* Implemented JWT Authentication using Passport.js
* Created authentication routes:

  * Register user
  * Login user
* Built Ferry module with:

  * Create ferry
  * Get all ferries
  * Get ferry by ID
  * Update ferry
  * Delete ferry
* Structured project using MVC architecture:

  * Routes
  * Controllers
  * Models

---

## at I learned

* How JWT authentication works
* How to structure backend using MVC pattern
* How to connect Node.js with MySQL
* How to write clean controller and model logic
* Importance of validation and error handling
